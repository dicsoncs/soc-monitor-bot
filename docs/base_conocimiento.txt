SOC:
Service Operations Center.
Centro encargado del monitoreo, gestión de eventos, seguimiento de incidencias y continuidad operacional de la red GPON.

TAC:
Technical Assistance Center.
Equipo encargado de la atención técnica especializada y soporte avanzado.

NOC:
Network Operations Center.
Área responsable del monitoreo y operación de la red de telecomunicaciones.

TCM:
Equipo encargado de la atención y seguimiento de incidencias técnicas.

GPON:
Gigabit Passive Optical Network.
Tecnología utilizada para la prestación de servicios de fibra óptica mediante una red pasiva.

OLT:
Optical Line Terminal.
Equipo principal de la red GPON encargado de administrar las ONT y ONU conectadas.

ONU:
Optical Network Unit.
Equipo instalado en el cliente final para recibir la señal GPON.

ONT:
Optical Network Terminal.
Dispositivo que convierte la señal óptica en servicios para el cliente.

NAP:
Network Access Point.
Punto de distribución donde se conectan múltiples clientes de la red GPON.

LOS:
Loss Of Signal.
Pérdida de señal óptica detectada entre la OLT y la ONU.

LOFI:
Loss Of Frame Indicator.
Alarma relacionada a pérdida de sincronismo de trama.

NCE:
Network Cloud Engine.
Plataforma utilizada para la administración y monitoreo de la red GPON.

HELIX:
Herramienta utilizada para la gestión de incidencias, tickets y seguimiento operativo.

SMARTWIFI:
Plataforma utilizada para monitoreo y validación de servicios GPON residenciales.

FAN SHARING:
Plataforma utilizada para monitoreo y validación de clientes GPON.

CAIDA DE OLT:
1. Validar alarmas.
2. Revisar energía.
3. Verificar uplinks.
4. Escalar a TCM.
5. Informar a NOC.

CAIDA DE NAP:
1. Validar clientes afectados.
2. Identificar OLT asociada.
3. Registrar ticket.
4. Escalar a TCM.

VALIDACION DE POTENCIA:
1. Ingresar al servicio.
2. ONU Optical Module Info.
3. Revisar Rx Optical Power.
4. Valor recomendado hasta -23 dBm.

EVENTO MASIVO:
Incidente que afecta múltiples clientes simultáneamente.

ESCALAMIENTO MASIVO:
1. Validar impacto del evento.
2. Identificar OLT afectada.
3. Registrar ticket.
4. Escalar a TCM.
5. Informar a NOC.
6. Realizar seguimiento hasta la normalización.

CLIENTES AFECTADOS:
Cantidad de usuarios impactados por una incidencia.

PRUEBA:
Esto es una prueba del SOC Assistant.

AAA:
Plataforma encargada de autenticar dispositivos o usuarios, autorizar el acceso a los servicios de red y registrar la información de las sesiones.

PROVISION AAA:
Permite registrar suscriptores.
Permite dar de baja suscriptores.
Permite modificar estado del usuario.
Permite modificar perfiles de velocidad.

ACS GENIE:
Plataforma de gestión remota de equipos CPE (ONT y routers) basada en TR-069.

BUSCAR SERIAL ACS:
1. Ingresar a ACS Genie.
2. Buscar por Serial Number.
3. Revisar detalle del dispositivo.

VALIDAR ONT ACS:
1. Buscar la ONT por Serial.
2. Verificar estado de comunicación.
3. Validar IP de gestión.
4. Confirmar si se encuentra online.

ACTUALIZAR FIRMWARE:
1. Buscar la ONT.
2. Ejecutar Push File.
3. Seleccionar firmware homologado.
4. Verificar resultado.

FAN SHARING:
Arquitectura de interconexión que permite compartir infraestructura entre Entel y WINET para servicios GPON.

VALIDAR FAN SHARING:
1. Ingresar a Resource List.
2. Buscar ONT SN.
3. Revisar estado del servicio.
4. Verificar OLT.
5. Verificar PON Port.
6. Verificar ONT ID.

BROADSOFT:
Plataforma utilizada para gestión de servicios SIP y comunicaciones unificadas.

BROADWORKS:
Plataforma que integra voz, correo, video y servicios SIP.

PROVISION SIP:
1. Ingresar a BroadWorks.
2. Crear Empresa.
3. Crear Grupo.
4. Crear Usuario.
5. Configurar Address.
6. Configurar autenticación.

HUNT GROUP:
Servicio utilizado para distribuir llamadas entre múltiples líneas SIP.

CALL PICKUP:
Permite capturar llamadas dentro de un grupo.

CORTE SIP:
Permite realizar corte parcial o total de una línea SIP.
